# workfast-ai-new

> Built with **Surge AI** — React + TypeScript + Vite + Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Tech Stack

| Tool | Version |
|------|---------|
| React | 18.x |
| TypeScript | 5.x |
| Vite | 5.x |
| Tailwind CSS | 3.x |
| React Router | 6.x |
| TanStack Query | 5.x |
| lucide-react | 0.x |

## Project Structure

```
src/
├── components/     # Reusable UI components
│   └── ui/         # Base UI primitives
├── hooks/          # Custom React hooks
├── lib/            # Utility functions
├── pages/          # Page components
├── App.tsx         # Root component
├── main.tsx        # Entry point
└── index.css       # Global styles
```
